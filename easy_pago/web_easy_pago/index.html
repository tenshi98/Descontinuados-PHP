Convirtiendo la Bodega un centro de pago

EasyPago es una aplicación diseñada exclusivamente para mini-mercados almacenes de barrio y/o bodegas. A través de un Smartphone, el usuario (locatario) puede ingresar de forma táctil e intuitiva la información requerida para el proceso como sería la empresa, el número de cliente y los montos de pago. Este proceso se realiza a través de Internet.

De forma similar, para locales que no posean este tipo de conexión, pueden a través de la telefonía tradicional, realizar los mismos procesos, sin embargo, para ello el usuario debe escuchar atento los mensajes recibidos por el IVR y digitar los números que el sistema solicita.
