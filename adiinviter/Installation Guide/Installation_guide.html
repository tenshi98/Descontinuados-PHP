Introduction
This guide will help you through the installation process and get AdiInviter Pro running on your website.
Before you begin, you will need an FTP client, if you do not have one, some popular solutions include FileZilla (free) or CuteFTP (trial).
This guide assumes that you have successfully downloaded the latest release of AdiInviter Pro and have the zip file "unzipped" and ready to go. If not, you can download the package from our customer downloads section.
The instructions will term the zip file you have downloaded as "adiInviter_pro_v1_1_standalone.zip".
Developer Recommendations
1. This guide is best viewed in Google Chrome browser (in Windows). Mac users, you won't find a difference.
2. We recommend Sublime Text Editor for coding and editing files because we use it too and its really awesome.
3. Before You proceed with the installation, you might wanna grab yourself a cup of coffee at this moment.
Uploading
Uploading
At this point, you should have the zip archive "adiInviter_pro_v1_1_standalone.zip".
Using your FTP client, upload/copy the contents of "Upload" folder inside product zip to your website root (Do not copy "Upload" folder itself).
The "webroot" directory is usually 'public_html' or 'www', but this varies from server to server so if you're unsure, contact your hosting provider.
You can upload the contents in any sub directory on your server. Please make sure to follow your sub directory upload URL while referring to this documentation.
Note - After uploading files to your server, assign "Write Permission" to following file on your server -
root/find_friends/adi_admincp/config.php
Introduction Installation
Installation
At this point, you should have uploaded AdiInviter Pro package files to your server.
Step 1
Visit following URL in your web browser (replace yourdomain.com with your website address) -
http://www.yourdomain.com/find_friends/adi_admincp/index.php
You will be asked to set "Username" and "Password" for AdiInviter Pro Admin Control Panel.
Step 2
In the next step, you will be asked to enter "Database Connection Details".
Note - Enter connection details for your website database. Do not create a new or separate database for AdiInviter Pro.
If the installation was completed successfully, you will see a message saying "AdiInviter Pro installed successfully" .
Step 3
Using your FTP client, remove the installer file located at following location on your server -
http://www.yourdomain.com/find_friends/adi_admincp/db_installer.php
Uploading Integration
Integration
Login to AdiInviter Pro Admin Control Panel and go to "DATABASE INTEGRATION" tab. Specify database table details for various systems in your website.
Note - If you do not have any of the systems in your website under "DATABASE INTEGRATION" tab, please leave their fields blank (For e.g. User System, Usergroup System, Friends/Followers/Subscribers system etc).
If you have specified "User Table Details" under "DATABASE INTEGRATION" tab, then perform following steps -
1. Using your FTP client, download the following file from your server and open it in any text editor -
http://www.yourdomain.com/find_friends/adiinviter_platform_settings.php
2. Search for following PHP variable -
$default_usergroupid
Set above variable to not loggedin or guest usergroup id in your website. By default it is set to 0 (zero).
3. Search for following PHP function in "adiinviter_platform_settings.php" file -
get_loggedin_userid()
Write your own code under "get_loggedin_userid()" function to return currently loggedin user's userid in your website. For not loggedin users, return the value 0 (zero).
Installation Configuration
Configuration
Login to AdiInviter Pro Admin Control Panel, go to "GLOBAL SETTINGS" tab and specify your website details.
Go to "INVITATION MAIL" tab. Customize various invitation templates to suit your website design and niche.
Setup OAuth Applications
Refer to following steps to setup and obtain OAuth application keys for required services.
Gmail
1. Visit following URL in your web browser and login with your Google account.
https://console.developers.google.com/project
2. Click on "Create Project" button and specify "Project Name" (This can be your website name) and "Project Id". Click on "Create" button.
3. Click on your project. Go to "APIS & AUTH" menu under left navigation bar (Located at extreme left side). Under "APIS & AUTH", click on "APIs" menu.
4. You will see a huge list of Google APIs. Look for "Contacts API" there and turn it ON (ON/OFF button is located at extreme right side).
5. Under "APIS & AUTH" menu (Located at extreme left side), click on "Credentials" menu. Click on "Create new Client ID" button and specify following settings -
- Set "APPLICATION TYPE" to "Web application".
- Ignore "AUTHORIZED JAVASCRIPT ORIGINS" option.
- Under "AUTHORIZED REDIRECT URI", specify following URIs (replace yourdomain.com with your website address) -
http://www.yourdomain.com/find_friends/adi_js.php?adi_do=oauth_login&adi_service=gmail
http://yourdomain.com/find_friends/adi_js.php?adi_do=oauth_login&adi_service=Gmail
Note - Please note that "http://" or "https://" is required in above URIs. We are specifying your website URIs with and without www. Enter one URI per line.
- Click on "Create Client ID" button.
6. Under "Client ID for web application" details, copy "CLIENT ID" and "CLIENT SECRET" (Make sure you are not selecting any blank spaces around those keys while copying).
Go to "OAUTH APPLICATIONS TAB" under AdiInviter Pro Admin Control Panel and specify above keys under Google Services "Client ID" and "Client Secret".
7. Lets go back to Google Developers Console (Where we created above application) and go to "Consent Screen" menu under "APIS & AUTH" menu (Located at extreme left side). Specify following details there -
- Specify your email address under "EMAIL ADDRESS" field.
- "PRODUCT NAME" can be your website name.
- Specify "HOMEPAGE URL" and "LOGO".
- Specify working URLs under "PRIVACY POLICY URL" and "TERMS OF SERVICE URL" even though they are optional. If you specify wrong or blank URLs here, the application might not work properly.
- You can ignore "GOOGLE+ PAGE" option if your website does not have page on Google+.
- Click on "Save" button.
Yahoo
1. Visit following URL in your web browser and login with your Yahoo account.
https://developer.apps.yahoo.com/dashboard/createKey.html
Specify following settings there -
- "Application Name" can be your website name.
- Set "Application Type" option to "Web-based".
- Write short description about your website under "Description" field.
- Specify "Home Page URL" as your website's domain URL.
- Under "Access Scopes" option, choose "This app requires access to private user data".
- Under "Callback Domain" field, specify your domain URL. You can enter same value as "Home Page URL".
- Under "Select APIs for private user data access", check/tick "Contacts" and choose "Read" option.
- Do not check/tick "Premium API services" option.
- Check/Tick "I have read and agree to the Yahoo! Developer Network Terms of Use." and click on "Create Project" button.
2. Copy "Consumer Key" and "Consumer Secret" (You have to click and then copy these keys. Make sure there are no dots "..." in your keys).
Go to "OAUTH APPLICATIONS TAB" under AdiInviter Pro Admin Control Panel and specify above keys under Yahoo "Consumer Key" and "Consumer Secret" fields.
3. Lets go back to Yahoo Developer Network (Where we created above application). Click on "Verify Domain" button next to your "Callback Domain".
Click on "Verify" button next to your domain and follow the instructions provided in popup.
Outlook, Hotmail, Live, MSN
1. Visit following URL in your web browser and login with your outlook account.
https://account.live.com/developers/applications/index
Click on "Create application" link. Specify "Application name" (this can be your website name) and click on "I accept" button.
Specify all Basic Settings which are pretty much self explanatory. Make sure to provide working URLs for "Terms of service URL" and "Privacy URL". If you provide blank or non-working URLs, the application might not work.

http://www.yourdomain.com/find_friends/hotmail_redirect.php

3. Go to "App Settings" under left navigation menu and copy "Client ID" & "Client secret (v1)" keys. Make sure you are not selecting any blank spaces around those keys while copying.
Go to "OAUTH APPLICATIONS TAB" under AdiInviter Pro Admin Control Panel and specify above keys under Outlook "Client ID" and "Client secret (v1)" fields.
Twitter

- Under "Callback URL" field, specify following URL replacing yourdomain.com with your website address -
http://www.yourdomain.com/find_friends/adi_js.php?adi_do=oauth_login&adi_service=twitter

2. Go to "Permissions" tab and set "Access" setting to "Read and Write". Click on "Update settings" button.

Go to "OAUTH APPLICATIONS TAB" under AdiInviter Pro Admin Control Panel and specify above keys under Twitter "API key" and "API secret" fields.
Linkedin
1. Visit following URL in your web browser and login with your Linkedin account.
https://www.linkedin.com/secure/developer?newapp=s

- Under "Application Info" settings, Specify "Application Name" (this can be your website name).

- Specify "Contact Info" settings according to your requirements.
- For "Default Scope" setting under "OAuth User Agreement" settings, check/tick "r_basicprofile", "r_network", "w_messages"
- For "OAuth 2.0 Redirect URLs", "OAuth 1.0 Accept Redirect URL" and "OAuth 1.0 Cancel Redirect URL", specify following URL replacing yourdomain.com with your website address -
http://www.yourdomain.com/find_friends/adi_js.php?adi_do=oauth_login&adi_service=linkedin

2. Click on your application and copy "API Key" & "Secret Key" specified under "OAuth Keys". Make sure you are not selecting any blank spaces around those keys while copying.

Integration Run
Embed and Run
At this point, you have successfully finished installing, integrating and configuring AdiInviter Pro for your website.
You may want to take a look at various system configuration tabs under AdiInviter Pro Admin Control Panel.
Go to "GLOBAL SETTINGS" tab under Admin Control Panel and turn "ON" AdiInviter Pro system.
Following instructions will show you how to embed and run various AdiInviter Pro interfaces into your website -
Initial Modification
1. If you do not have <meta> tag added to your web page, add following line under <head> tag of your web page -
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
Note - If you already have <meta> tag added to your web page, then make sure "charset" is set to "utf-8".
2. Add following code under <head> tag of your web page -
<link rel="stylesheet" type="text/css" href="/find_friends/adi_css.php">
<script type="text/javascript" src="/find_friends/adiinviter/js/jquery.min.js"></script>
<script type="text/javascript" src="/find_friends/adiinviter/js/adiinviter.js"></script>
<script type="text/javascript" src="/find_friends/adi_params.php"></script>
Caveats -
If you have uploaded AdiInviter Pro under sub directory (not your website root like "public_html" or "www"), then update "src" URLs in above code.
If you already have jQuery 1.4 or above included in your web page, perform following steps -
- Remove second line from above code (highlighted in dark orange color).
- Add above code after jQuery library loading </script> tag.
Popup Model
You can access default AdiInviter Pro popup interface at following location in your website.
http://www.yourdomain.com/adiinviter_popup.php
To embed popup interface model into your webpage, perform the following steps -
1. Make sure you have done initial modification stated above.
2. Use any of the following codes to open popup interface in your webpage -
- To open default popup -
<a href="#" class="adi_open_popup_model">Find/Invite Friends</a>
- To open popup interface with default service Gmail (you can replace "gmail" in below code with any other service id) -
<a href="#" class="adi_open_popup_model" adi_service_id="gmail">Find/Invite Friends</a>
Inpage Model
You can access default AdiInviter Pro inpage interface at following location in your website.
http://www.yourdomain.com/adiinviter.php
To embed inpage interface model into your webpage, perform the following steps -
1. Make sure you have done initial modification stated above.
2. Add following PHP code in your web page -
<?php
if(!headers_sent())
{
  include_once('find_friends/adi_inpage.php');
}
?>
Note - If you have uploaded AdiInviter Pro under sub directory on your server (not your website root like "public_html" or "www"), then update "include_once" path in above code.
Note - Make sure to add above PHP code before anything is printed on your web page (i.e. before any HTTP headers are sent).
3. Add following PHP code in your web page where your want to print AdiInviter Pro Inpage Model interface -
<?php
echo $contents;
?>
